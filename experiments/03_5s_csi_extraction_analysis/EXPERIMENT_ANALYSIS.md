# 5s CSI Extraction Analysis

## Objective

Validate that the 5-second dual-RX capture can be converted into usable CSI and
inspect three first-order features:

```text
CSI amplitude heatmap
Mean CSI amplitude over time
Relative RX phase over time
```

## Input Data

```text
data/test_rx_5s/H.npy
data/test_rx_5s/csi_info.json
data/test_rx_5s/capture_config.json
data/test_rx_5s/probe_metadata.json
```

The extracted CSI tensor has shape:

```text
H.shape = (4993, 2, 2, 52)
layout = [frame, rx, tx, active_carrier]
```

This means:

```text
4993 frames
2 RX channels
2 TX channels
52 active OFDM subcarriers
```

With `probe_rate = 1000 Hz`, the extracted CSI covers about:

```text
4993 / 1000 = 4.993 s
```

The expected duration was 5 seconds, so frame extraction is effectively
complete.

## Amplitude Heatmap

The amplitude heatmaps show a stable CSI structure across the 5-second capture.
The four MIMO links have different average amplitudes, which is expected due to
different RF paths, antenna coupling, channel gains, and hardware chain gains.

Mean amplitude by link:

```text
RX0<-TX0: -43.58 dB
RX0<-TX1: -48.13 dB
RX1<-TX0: -42.89 dB
RX1<-TX1: -40.45 dB
```

Amplitude standard deviation by link:

```text
RX0<-TX0: 4.07 dB
RX0<-TX1: 5.39 dB
RX1<-TX0: 2.57 dB
RX1<-TX1: 1.90 dB
```

The strongest and most stable link is `RX1<-TX1`. The weakest and most variable
link is `RX0<-TX1`.

## Mean Amplitude Over Time

The mean-amplitude time series is mostly stable over the 5-second interval.
This indicates that the frame extraction is aligned and the pilot-based CSI
calculation is not randomly drifting frame to frame.

The link ordering is consistent:

```text
strongest: RX1<-TX1
middle: RX1<-TX0 and RX0<-TX0
weakest: RX0<-TX1
```

There is no large abrupt amplitude jump in this short capture. For sensing, this
means the current capture is a good baseline or static-channel reference.

## Relative RX Phase

The relative phase plot uses:

```text
angle(H[:, 0, tx, :] * conj(H[:, 1, tx, :]))
```

This removes part of the common TX-side phase and is usually more useful than
absolute single-link phase.

Observed behavior:

```text
TX0 relative phase changes within roughly several radians.
TX1 relative phase has stronger slow variation than TX0.
The phase traces are structured rather than purely random.
```

This suggests that relative RX phase can be used as a sensing feature, but it
should be filtered or detrended before downstream use.

## Adjacent-Frame Stability

Adjacent-frame stability was evaluated by comparing `H[t]` and `H[t-1]`.

Summary metrics:

```text
complex correlation mean: 0.743
complex correlation median: 0.775
complex correlation 5th percentile: 0.628
relative complex change mean: 1.128
relative complex change median: 1.124
mean absolute amplitude step: 3.73 dB
median absolute amplitude step: 3.72 dB
95th percentile amplitude step: 4.13 dB
mean absolute phase step: 1.18 rad
median absolute phase step: 1.17 rad
95th percentile phase step: 1.49 rad
```

Per-link mean absolute amplitude step:

```text
RX0<-TX0: 4.30 dB
RX0<-TX1: 5.78 dB
RX1<-TX0: 2.76 dB
RX1<-TX1: 2.08 dB
```

Per-link mean absolute phase step:

```text
RX0<-TX0: 1.15 rad
RX0<-TX1: 1.40 rad
RX1<-TX0: 1.08 rad
RX1<-TX1: 1.08 rad
```

Interpretation:

```text
The raw complex CSI is not frame-to-frame constant.
Adjacent complex correlation is moderate, not near 1.
Amplitude is more stable on RX1 links than RX0 links.
RX1<-TX1 is the most stable link by amplitude step.
RX0<-TX1 is the least stable link by amplitude step and phase step.
```

The stability plot also shows a drop in complex correlation near the end of the
capture. Since the amplitude step does not show a matching catastrophic jump,
this is likely dominated by residual phase/synchronization variation rather
than a complete loss of received signal.

For downstream sensing, do not directly compare raw complex `H` frame by frame
without preprocessing. Prefer:

```text
smoothed amplitude features
relative RX phase features
detrended phase features
subcarrier-averaged features
PCA or low-pass filtered CSI features
```

## Interpretation

The extracted `H.npy` is usable for basic CSI sensing experiments:

```text
frame extraction is near-complete
all 4 MIMO links are populated
52 active subcarriers are available per frame
amplitude is stable enough for baseline analysis
relative phase has observable temporal structure
raw adjacent-frame complex CSI still has noticeable jitter
```

Recommended next analyses:

```text
Apply smoothing to mean amplitude and relative phase.
Compute PCA over amplitude or phase-difference features.
Compare static baseline against motion captures.
Compute Doppler-like spectra from CSI time series.
Analyze per-subcarrier variance to select stable or sensitive subcarriers.
```

## Caveats

The current capture is only 5 seconds. It is enough to validate extraction but
not enough to characterize long-term drift or sensing robustness.

Absolute phase should not be used directly without calibration. Prefer relative
phase, phase differences, or detrended phase features.
